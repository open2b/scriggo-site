Scriggo

Build, customize and extend applications using Go as a scripting language

Scriggo is the fast and safe open-source Go runtime to embed the Go language as a scripting language in Go applications and templates.

Get Started

Implement new features faster

Implement and try new features faster in your application without modifying its original code base.

You can move the code into the original code base only when it is stabilized.

Respond rapidly to changes

The parts of your application that require more frequent changes to adapt to the context can be changed in seconds, no hours.

No need to recompile and restart the application.

Involve non-programmers

Scriggo can allow non programmers, as marketers, to easy code well-defined application behaviors directly in the application.

The Go language with its simple syntax is easy to learn for non-programmers.

Keep programming in Go

Scriggo does not require learning a new language. Keep to code with the language you love. Use the Go standard library, your existing code and any Go module.

Fast compilation and execution

Scriggo instantly compiles the source code to its virtual machine code. Then each program execution is as fast as mainstream languages like Ruby, Python and Lua.

Run untrusted code safely

Executed code can have access only to packages, values and types that you explicitly provided. You can limit also the consumed memory and the total time of execution.

Make Go modules scriptable

Let Scriggo create an interpreter for a Go module and distribute it to users as a standalone executable.

They can try the module's functions without having to worry about download and install Go and the required modules.

Fast and powerful template engine

The included Scriggo template engine is designed from the ground up to fully support Go in templates.

It's modern and safe. Supports inheritance, macros, includes, imports and autoescaping.
